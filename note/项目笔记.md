vue init webpack <项目名称>

2. 安装依赖

进入第1步中初始化的项目目录, 执行命令:

```bash
npm install
```



3. 手动安装vuex

```bash
npm install vuex --save
```



4. 目录说明

```bash
src
├── App.vue
├── assets        -> 存放样式和图片
│   ├── css
│   ├── img
│   └── logo.png
├── components    -> 存放公共组件
│   └── HelloWorld.vue
├── main.js
├── router        -> 存放路由
│   └── index.js
├── store         -> 存放vuex
└── views         -> 存放单页面
```
